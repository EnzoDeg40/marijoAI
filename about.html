What is a Neural Network?

A neural network is a computational model inspired by the structure and function of biological neural networks in the human brain. It consists of interconnected nodes (neurons) organized in layers that process information to make predictions or decisions.

Neural Network Architecture

How Neural Networks Work

1. Input Layer

Receives raw data (features) and passes it to the first hidden layer. Each neuron represents one feature from your dataset.

2. Hidden Layers

Perform complex transformations on the data. Each neuron applies weights, biases, and activation functions to create new feature representations.

3. Output Layer

Produces the final prediction or classification result. The number of neurons depends on your problem type (1 for regression, multiple for classification).

4. Weights & Biases

Parameters that the network learns during training. Weights control the influence of each connection, biases shift the activation threshold.

Key Training Concepts

Epochs

One epoch is a full pass through the entire training dataset. More epochs allow the model to learn more patterns, but too many can lead to overfitting (when the model learns the training data too well and performs poorly on new data). Monitor loss and accuracy to choose a reasonable number.

Batch Size

The number of samples processed before the model updates its weights. Smaller batches can help generalization but add noise; larger batches are smoother and faster on modern hardware but require more memory.

Learning Rate

Controls the step size of each weight update. If it is too high, training can diverge or bounce; if too low, training can be very slow. Start small and adjust based on the loss curve.

Normalization

Rescaling input features to a similar range (e.g. 0-1). Normalized inputs make training more stable and faster. Prediction data should be normalized the same way.

Activation Functions

Activation functions are mathematical operations applied to the output of each neuron. They introduce non-linearity into the network, allowing it to learn complex patterns. Different activation functions have different characteristics and are suited for different types of problems.

ReLU (Rectified Linear Unit)

Formula: f(x) = max(0, x)

Use cases: Most popular choice for hidden layers. Simple, efficient, and helps with vanishing gradient problem.

Sigmoid

Formula: f(x) = 1 / (1 + e^(-x))

Use cases: Binary classification, output layer when probabilities are needed.

Tanh (Hyperbolic Tangent)

Formula: f(x) = tanh(x)

Use cases: Hidden layers, especially in recurrent networks.

Softmax

Use cases: Multi-class classification output layer. Converts outputs to probabilities that sum to 1.

Linear

Formula: f(x) = x

Use cases: Regression problems, output layer for continuous values. No transformation applied.

Use cases: Modern alternative to ReLU, smooth and non-monotonic.

Common Use Cases

Neural networks can solve a wide variety of problems across different industries and domains:

Classification Problems

Examples: Email spam detection, image recognition, medical diagnosis, sentiment analysis

Output: Categorical predictions (e.g., "spam" vs "not spam")

Activation: Sigmoid for binary, Softmax for multi-class

Regression Problems

Examples: House price prediction, stock price forecasting, temperature prediction

Output: Continuous numerical values

Activation: Linear for the output layer

Pattern Recognition

Examples: Fraud detection, anomaly detection, quality control

Output: Binary or multi-class classifications

Activation: ReLU or Swish for hidden layers

Recommendation Systems

Examples: Product recommendations, content suggestions, personalized marketing

Output: Probability scores or rankings

Activation: Sigmoid or Softmax for output
