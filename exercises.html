Before You Start

Download the Wisconsin breast cancer dataset, and clean it using the Clean Data page (select 'yes' to the question about the Header row, and keep the other Cleaning Options as default).

Then, split it into a training and a validation sets, keeping "Validation Split" at the default value (this will be available after cleaning).

Download the prepared datasets and use them throughout the exercises below.

1

Building Your First Breast Cancer Classifier

Objective: Build a neural network to classify breast cancer as benign or malignant

Tasks:

  1. Examine the datasets (30 features + 1 diagnosis column + 1 ID column)
  2. Create a neural network with the following exact architecture:
    • Input Layer: 30 neurons (one for each feature)
    • Hidden Layer 1: 16 neurons with ReLU activation
    • Hidden Layer 2: 8 neurons with ReLU activation
    • Output Layer: 1 neuron with Sigmoid activation (for binary classification)
  3. Train the model you created in the Train page, using the default settings, passing the training dataset you downloaded after cleaning as training data, and selecting "Diagnosis" as the target.

Reflection Questions:

  • What is a Neural Network?.
  • Why was the Sigmoid activation function used for the output layer?
2

Exploring Activation Functions

Objective: Compare different activation functions and their effects on breast cancer classification

Tasks:

  1. Create 3 identical neural networks with this architecture:
    • Input: 30 neurons
    • Hidden: 12 neurons (the activation function will vary)
    • Output: 1 neuron with Sigmoid
  2. Model 1: Use ReLU for hidden layer
  3. Model 2: Use Sigmoid for hidden layer
  4. Model 3: Use Tanh for hidden layer
  5. Train all three with the default settings
  6. Compare their training performance and final accuracy

Reflection Questions:

  • Which activation function performed best?
  • From what you see, what are the advantages and disadvantages of each activation function?
3

Learning Rate Experiment

Objective: Understand how learning rate affects training on breast cancer classification

Tasks:

  1. Create a neural network with this architecture:
    • Input: 30 neurons
    • Hidden: 10 neurons with ReLU
    • Output: 1 neuron with Sigmoid
  2. Train the same model 3 times with different learning rates: 0.01, 0.001, and 0.0001
  3. Continue experimenting with different learning rates
  4. Keep the other hyperparameters at default for all experiments
  5. Record the final training loss and accuracy for each learning rate

Reflection Questions:

  • What happens when the learning rate is too low?
  • What happens when the learning rate is too high?
4

Overfitting Investigation

Objective: Learn to identify and prevent overfitting in breast cancer classification

Tasks:

  1. Create a complex neural network with this architecture:
    • Input: 30 neurons
    • Hidden Layer 1: 64 neurons with ReLU
    • Hidden Layer 2: 32 neurons with ReLU
    • Hidden Layer 3: 16 neurons with ReLU
    • Output: 1 neuron with Sigmoid
  2. Train for 200 epochs with the other hyperparameters at default
  3. Monitor both training loss AND validation loss
  4. Enable early stopping with patience=10 and train the Model again

Reflection Questions:

  • What is overfitting?
  • How does early stopping help prevent overfitting?
5

Data Normalization Impact

Objective: Learn why data normalization is important

Tasks:

  1. Download the dataset and do the cleaning again, but this time, don't normalize the data.
  2. Train a model without normalization
  3. Train an identical model with normalization
  4. Compare the final accuracy

Reflection Questions:

  • How did normalization improve training?
  • When is normalization most important?
6

Batch Size Experiment

Objective: Understand how batch size affects training

Tasks:

  1. Train the same model with batch sizes: 8, 16, 32, 64
  2. Keep all other parameters identical
  3. Record the final training loss and accuracy for each batch size
  4. Analyze the trade-offs between different batch sizes

Reflection Questions:

  • How does batch size affect training?
  • What are the advantages of smaller vs larger batch sizes?

Real-World Prediction Project

Objective: Apply all learned concepts to a complete breast cancer prediction project

Tasks:

  1. Design and train multiple neural network architectures
  2. Experiment with different hyperparameters (learning rate, batch size, epochs)
  3. Compare all models and select the best one based on performance metrics
  4. Make predictions on validation data and evaluate performance
  5. Document your findings and explain which approach worked best

Reflection Questions:

  • What challenges did you face in creating the best Model architecture possible?
  • How could this breast cancer prediction model be used in real-world medical applications?
